Mutation 2009 Program

09:00 - 09:10 Welcome

09:10 - 10:10 Keynote

  • Jeff Offutt: A Mutation Carol: Past, Present and Future [slides]

10:10 - 10:30 Session 1 - Mutation analysis with aspects

  • AjMutator: A Tool For The Mutation Analysis Of AspectJ Pointcut (Romain Delamare, Benoit Baudry, Yves Le Traon) [slides]

10:30 - 11:00 Coffee Break

11:00 - 12:30 Session 2 - Mutation testing

  • Mutation Analysis of Parameterized Unit Tests (Tao Xie, Nikolai Tillmann, Jonathan de Halleux, Wolfram Schulte) [slides]
  • Comparison of Unit-Level Automated Test Generation Tools (Shuang Wang, Jeff Offutt) [slides]
  • Using Logic Criterion Feasibility to Reduce Test Set Size While Guaranteeing Double Fault Detection (Garrett Kaminski, Paul Ammann) [slides]

12:30 - 14:00 Lunch

14:00 - 15:30 Session 3 - Empirical studies

  • The Impact of Equivalent Mutants (Bernhard Grün, David Schuler, Andreas Zeller) [slides]
  • An Experimental Comparison of Four Unit Test Criteria: Mutation, Edge-Pair, All-uses and Prime Path Coverage (Nan Li, Upsorn Praphamontripong, Jeff Offutt) [slides]
  • Advantages of mutation in passive testing: An empirical study (Cesar Andres, Mercedes Merayo, Carlos Molinero) [slides]

15:30 - 16:00 Coffee Break

16:00 - 17:30 Session 4 - New mutation applications

  • A Mutation / Injection-based Automatic Framework for Evaluating Code Clone Detection Tools (Chanchal Kumar Roy, James R. Cordy) [slides]
  • Assertion-Driven Development: Assessing the Quality of Contracts using Meta-Mutations (Thomas Knauth, Christof Fetzer, Pascal Felber) [slides]
  • Evaluating testing strategies for imaging software by means of Mutation Analysis (Rene Just, Franz Schweiggert) [slides]

17:30 Closing
