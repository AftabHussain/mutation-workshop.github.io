Program

April 17, 2012

Time Title
09:00 - 09:15 Opening
09:15 - 10:30 Keynote

Gordon Fraser: Easing the Oracle Problem with Mutation Testing.
10:30 - 11:00 Coffee break
11:00 - 12:30 Paper session 1 - Efficiency of mutation

- Vinicius Durelli, Jeff Offutt and Marcio Delamaro. Toward Harnessing High-level Language Virtual Machines for Further Speeding up Weak Mutation Testing.

- Pedro Reales Mateo and Macario Polo Usaola. Mutant Execution Cost Reduction.

- Rene Just, Gregory Kapfhammer and Franz Schweiggert. Do Redundant Mutants Affect the Effectiveness and Efficiency of Mutation Analysis?

12:30 - 14:00 Lunch
14:00 - 15:30 Paper session 2 - Novel mutation testing paradigms

- Marinos Kintis, Mike Papadakis and Nicos Malevris. Isolating First Order Equivalent Mutants via Second Order Mutation.

- Haitao Dan and Rob Hierons. SMT-C: A Semantic Mutation Testing Tool for C.

- Matthew Patrick, Manuel Oriol and John Clark. MESSI: Mutant Evaluation by Static Semantic Interpretation.

15:30 - 16:00 Coffee break
16:00 - 1700 Paper session 3 - New applications

- Mike Papadakis and Yves Le Traon. Using mutants to locate “unknown” faults.

- Yehia Elrakaiby, Tejeddine Mouelhi and Yves Le Traon. Testing Obligation Policy Enforcement using Mutation Testing.
